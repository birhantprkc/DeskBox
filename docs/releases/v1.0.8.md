# DeskBox 1.0.8

## English

DeskBox 1.0.8 is a focused compatibility and polish release after 1.0.7. It improves Windows 11 23H2 drag/drop behavior, clarifies mapped-folder drag prompts, and makes Quick Capture copy feedback more stable and reliable.

### Download

- [DeskBox_Setup_1.0.8_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.8/DeskBox_Setup_1.0.8_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Better Windows 11 23H2 drag/drop compatibility: the post-install launch now runs as the original user, reducing Explorer-to-DeskBox drag permission issues.
- More tolerant Explorer drag handling: DeskBox accepts compatible link-style drag requests and resolves them into the configured move/copy behavior.
- Clearer drag prompts: managed storage widgets and mapped-folder widgets now show distinct hover captions.
- Stable Quick Capture copy feedback: click-to-copy now uses one bottom-centered toast instead of per-row bubbles.
- More reliable clipboard writes: Quick Capture retries briefly when Windows temporarily locks the clipboard, reducing first-click copy failures.
- More consistent controls: Quick Capture title buttons and item hover actions now better match regular widget button size and styling.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Quick Capture data, and managed storage content.

## 中文

DeskBox 1.0.8 是 1.0.7 之后的兼容性和细节优化版本。这个版本重点处理 Windows 11 23H2 下的拖拽兼容、映射文件夹拖拽提示，以及随记复制反馈不稳定的问题。

### 下载

- [DeskBox_Setup_1.0.8_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.8/DeskBox_Setup_1.0.8_x64.exe)

安装器会检测 .NET 8 Runtime x64 和 Windows App Runtime 2.1.3 x64。缺少依赖时，安装流程可以联网下载并安装。

### 主要更新

- 优化 Windows 11 23H2 拖拽兼容性：安装后启动 DeskBox 时回到原始用户权限，减少资源管理器向 DeskBox 拖拽时的权限阻断。
- 兼容更多资源管理器拖拽格式：对可安全处理的 link-style 拖拽请求，会按当前收纳设置转成移动或复制。
- 拖拽提示更清楚：收纳格子和映射文件夹会显示不同的悬浮提示，减少目标误解。
- 随记复制提示更稳定：单击复制统一使用底部居中的 toast，不再跟随每条记录左右漂移。
- 随记复制更可靠：当 Windows 剪贴板被短暂占用时自动重试，减少第一次单击复制失败。
- 控件样式更统一：随记右上角按钮和记录悬浮按钮尺寸/样式更接近普通格子。

### 兼容性

- 目标系统：Windows 11。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、组件配置、随记数据和收纳目录内容。
