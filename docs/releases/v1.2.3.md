# DeskBox 1.2.3

## English

DeskBox 1.2.3 is a small stability and UI polish release after the .NET 10 / Windows App SDK 2.2 baseline update. It focuses on the new desktop layer mode prototype and a few Settings visual details reported during real use.

### Download

- [DeskBox_Setup_1.2.3_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.2.3/DeskBox_Setup_1.2.3_x64.exe)

SHA256:

```text
17C029F907FE8D4C6B49DAC1A12DAA632C58E2B32537F51D1E17BBEF02EDEB35
```

The installer checks for .NET 10 Runtime x64 and Windows App Runtime 2.2 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Added a desktop widget layer mode setting in Settings -> General.
- Kept the existing Dynamic layer behavior and added a Desktop pinned option for users who want widgets to stay on the desktop layer after Show Desktop / Win+D.
- Improved desktop-layer refresh when the layer mode changes or when the display environment is refreshed.
- Updated Settings navigation icons to Fluent color icons.
- Removed redundant ToggleSwitch on/off text while keeping the native WinUI switch visual style.
- Removed extra decorative icons from the widget title icon setting and update-status rows.

### Notes For Testing

- Open Settings -> General and switch the desktop widget layer mode between Dynamic and Desktop pinned.
- In Desktop pinned mode, test Show Desktop / Win+D, tray show/hide, widget drag, resize, right-click menus, Todo editor, Quick Capture editor, and file widget context menus.
- Check Settings in light and dark themes and verify switches still use the native WinUI visual style.
- Overwrite install from 1.2.2 and confirm existing widgets, settings, Todo data, Quick Capture data, and managed storage content remain intact.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 10 Runtime x64 and Windows App Runtime 2.2 x64.
- Overwrite installs preserve existing app settings, widget configuration, Todo data, Quick Capture data, and managed storage content.

### 主要更新

- 在「设置 -> 常规」新增桌面格子层级模式。
- 保留现有动态层级行为，同时新增桌面固定层选项，适合希望格子在“显示桌面”/ Win+D 后仍停留在桌面层的用户。
- 优化层级模式切换和显示环境刷新时的桌面层恢复逻辑。
- 设置左侧导航图标换成 Fluent 彩色图标。
- 删除 ToggleSwitch 多余的开关文字，同时保留原生 WinUI 开关视觉。
- 删除“格子标题图标”和“更新状态”行里多余的装饰图标。

### 测试建议

- 打开「设置 -> 常规」，在动态层级和桌面固定层之间切换桌面格子层级模式。
- 在桌面固定层模式下测试“显示桌面”/ Win+D、托盘显示隐藏、格子拖拽、调整大小、右键菜单、待办编辑、随记编辑和文件格子菜单。
- 在浅色和深色主题下检查设置页，确认开关仍然保持原生 WinUI 样式。
- 从 1.2.2 覆盖安装，确认已有格子、设置、待办数据、随记数据和收纳目录内容都能保留。

### 兼容性

- 目标系统：Windows 11。
- 运行依赖：.NET 10 Runtime x64、Windows App Runtime 2.2 x64。
- 覆盖安装会保留现有应用设置、格子配置、待办数据、随记数据和收纳目录内容。
