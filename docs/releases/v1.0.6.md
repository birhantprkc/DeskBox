# DeskBox 1.0.6

## English

DeskBox 1.0.6 expands DeskBox from a file-focused desktop organizer into a lighter cross-app content helper. The headline feature is Quick Capture, an optional local-only feature widget for reusable text, links, screenshots, and recent clipboard content. This release also improves upload workflows, drag/drop behavior, widget refresh, window layering, and high-DPI layout polish.

### Download

- [DeskBox_Setup_1.0.6_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.6/DeskBox_Setup_1.0.6_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Quick Capture: add an optional feature widget for local text, links, screenshots, and recent clipboard content.
- Records, Pinned, and Recent: switch between saved notes, pinned reusable content, and recent clipboard captures.
- Better Quick Capture interactions: hover actions, compact current-view search, image thumbnails, drag-out support, and save-to-file-widget actions.
- Upload-friendly storage access: pin DeskBox storage to Quick Access, open storage from the tray, and keep folder shortcuts for file pickers.
- Cleaner drag/drop and clipboard behavior: file drags stay file-first, path copying is explicit, and DeskBox's own clipboard writes are ignored by Recent capture.
- Better file widget details: custom Explorer icon refresh, filename extension display controls, shortcut-arrow settings placement, and clearer migration progress/result feedback.
- Better window layering: tray/global-hotkey raised widgets now stay raised until the user clicks another app, and Settings can temporarily join that topmost layer.
- UI polish: high-DPI onboarding layout fixes, compact Quick Capture edit dialogs, target-widget refresh/highlight after saving, and theme-aligned Quick Capture styling.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Quick Capture data, and managed storage content.

## 中文

DeskBox 1.0.6 将 DeskBox 从偏文件整理的桌面工具，扩展为更轻量的跨应用内容助手。这个版本的核心是新增“随记”功能格子，用于本地保存常用文本、链接、截图和最近复制内容。同时也优化了上传入口、拖拽行为、格子刷新、窗口层级和高 DPI 布局。

### 下载

- [DeskBox_Setup_1.0.6_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.6/DeskBox_Setup_1.0.6_x64.exe)

安装器会检测 .NET 8 Runtime x64 和 Windows App Runtime 2.1.3 x64。缺少依赖时，安装流程可以联网下载并安装。

### 主要更新

- 随记：新增可选功能格子，用于本地保存文本、链接、截图和最近复制内容。
- 记录、固定、最近：可在保存内容、常用固定内容和最近复制内容之间切换。
- 随记交互优化：支持悬停操作、当前视图搜索、图片缩略图、拖出内容和保存到文件格子。
- 上传友好入口：收纳路径可固定到快速访问，可从托盘打开，并为文件选择器保留格子文件夹快捷方式。
- 拖拽和剪贴板更干净：文件拖拽优先保持文件格式，复制路径改为显式操作，并忽略 DeskBox 自己写入剪贴板造成的最近记录污染。
- 文件格子细节增强：支持资源管理器自定义图标刷新、文件后缀显示控制、快捷方式箭头设置归位，并补充迁移进度和结果反馈。
- 窗口层级优化：托盘或快捷键唤起后，格子会保持临时置顶，直到用户点击其他应用；此状态下打开设置页也会临时置顶。
- 界面细节优化：修复高 DPI 下的新手引导布局，随记编辑弹窗适配小窗口，保存到文件格子后刷新并高亮目标文件，随记主题色与 DeskBox 设置保持一致。

### 兼容性

- 目标系统：Windows 11。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、组件配置、随记数据和收纳目录内容。
