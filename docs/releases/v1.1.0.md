# DeskBox 1.1.0

## English

DeskBox 1.1.0 focuses on drag-and-drop recovery, installer cleanup, widget sorting, and Quick Capture editing. If Windows 10/11 cannot drag files into DeskBox widgets, open Settings and run the new drag-and-drop diagnostics repair first. If the issue remains, please follow the WeChat public account in the app's About page and leave a message, or open a GitHub issue with logs and reproduction details.

### Download

- [DeskBox_Setup_1.1.0_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.1.0/DeskBox_Setup_1.1.0_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Added Settings -> Drag-and-drop diagnostics with one-click repair for DeskBox compatibility flags, startup entries, and shortcuts.
- Improved Explorer drag/drop compatibility for managed and mapped widgets, including native shell message allowance, legacy shell format fallback, and diagnostic logs.
- Fixed widget sorting stability with natural name ordering and correct insertion when files are added under the current sort mode.
- Improved Quick Capture text editing: double-click saved text to edit inline, or use the context menu to edit in Notepad and sync changes back.
- Changed the default tray icon style to colorful for new installs and restored defaults.
- Improved first-run onboarding so it appears only after the first install launch, while remaining available from Settings.
- Improved install/uninstall behavior, including selectable install path, startup selection during setup, and optional removal of local app data during uninstall.

### Compatibility

- Target system: Windows 11. Windows 10 is still not the primary target, but this release includes additional Win10 drag/drop diagnostics and compatibility handling.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Quick Capture data, and managed storage content.

## 中文

DeskBox 1.1.0 重点优化拖拽修复、安装卸载、格子排序和随记编辑。如果 Win10/Win11 遇到文件拖不进 DeskBox 格子的问题，请先打开“设置”，使用新的“拖拽异常诊断”一键修复。如果仍有问题，可以扫码关注应用“关于”页里的公众号留言，或在 GitHub 提交 issue，并附上日志和复现方式。

### 下载

- [DeskBox_Setup_1.1.0_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.1.0/DeskBox_Setup_1.1.0_x64.exe)

安装器会检测 .NET 8 Runtime x64 和 Windows App Runtime 2.1.3 x64。缺少依赖时，安装流程可以联网下载并安装。

### 主要更新

- 新增“设置 -> 拖拽异常诊断”，可一键修复 DeskBox 的兼容性标记、启动项和快捷方式。
- 优化资源管理器拖拽兼容，收纳格子和映射格子支持更多原生 shell 拖拽消息、旧格式兜底和诊断日志。
- 修复格子排序稳定性，使用更接近 Windows 的自然名称排序，并确保新增文件按当前排序模式插入。
- 优化随记文本编辑：已保存文本双击进入随记内编辑；右键可在记事本中编辑并同步回随记。
- 新安装和恢复默认设置时，托盘图标默认改为彩色。
- 优化新用户引导，只在首次安装启动后自动出现，之后可在设置中手动查看。
- 优化安装和卸载体验，支持选择安装目录、安装时选择开机自启，卸载时可选择是否删除本地应用数据。

### 兼容性

- 目标系统：Windows 11。Windows 10 仍不是主要目标，但此版本补充了 Win10 拖拽诊断和兼容处理。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、格子配置、随记数据和收纳目录内容。
