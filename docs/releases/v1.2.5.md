# DeskBox 1.2.5

## English

DeskBox 1.2.5 is a feature and refinement release for the Todo, Quick Capture, Music, file-widget, title-bar, and update experiences. It keeps the 1.2 architecture stable while tightening the details that affect daily use.

### Highlights

- Todo now keeps due-time precision, supports native Windows reminder notifications, marks overdue items, sorts completed items below active items, and adds click-to-copy plus multi-select copying.
- Todo and Quick Capture can exchange text through drag and drop.
- Todo and Quick Capture tab styles can be configured separately.
- Widget hover buttons are configurable globally, with refreshed Fluent action icons and improved compact-title sizing.
- Resetting display/content title styles now clears per-widget overrides so widgets follow the global title style.
- Music volume control is now horizontal, light-theme friendly, closes on click-away, and aligns album ambience corners with the widget corner setting.
- File widgets can hide or show full-path hover tooltips from Settings.
- Direct and Microsoft Store update channels now use clearer channel-specific wording and behavior.

### 更新重点

- 待办保留截止时间的时分秒精度，支持 Windows 原生提醒通知、逾期标识、已完成排序、单击复制和多选复制。
- 待办和随记之间支持文本拖拽转换。
- 待办和随记的顶部切换样式可以分别配置。
- 格子右上角悬浮按钮支持全局配置，并替换为更清晰的 Fluent 图标；紧凑标题模式下图标尺寸也继续收敛。
- 重置展示/内容格子的标题样式时，会清除单个格子的覆盖值，让它们重新跟随全局标题样式。
- 音乐音量浮窗改为横向，适配浅色模式，支持点击空白区域关闭，封面氛围背景圆角会跟随格子圆角设置。
- 文件格子新增完整路径提示开关，可在设置中隐藏鼠标移入时的路径提示。
- Direct 和 Microsoft Store 更新渠道的文案与行为继续区分，减少用户对更新来源的困惑。
