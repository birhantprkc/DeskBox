# DeskBox 1.0.9

## English

DeskBox 1.0.9 is a broad polish release for Settings, widget interaction, Quick Capture, and daily Windows desktop behavior. It focuses on making DeskBox feel calmer, more native, and more predictable when managing files from the desktop, tray, and global hotkey.

### Download

- [DeskBox_Setup_1.0.9_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.9/DeskBox_Setup_1.0.9_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.
DeskBox now installs for the current user by default. Older Program Files administrator installs are migrated automatically to reduce Explorer drag/drop permission conflicts.

If Explorer drag/drop still shows a blocked cursor or cannot drop files into DeskBox, please check whether DeskBox is running as administrator. For normal use, DeskBox should run with the same non-elevated user permission level as Explorer; otherwise Windows may block file drops before DeskBox can receive them.

### Highlights

- Reworked Settings into a cleaner Windows-style structure with native ComboBox, NumberBox, toggle labels, drill-in rows, tray icon options, and a clearer Quick Capture settings entry.
- Added widget item sorting by name, size, item type, and date modified, with per-widget persistence and repeat-click ascending/descending behavior.
- Improved widget menus so title-bar actions and content-area actions are separated more clearly.
- Improved Quick Capture tab switching, title buttons, hover actions, copy feedback, and shared show/restore behavior with regular widgets.
- Improved drag/drop compatibility, empty-widget drop handling, managed-vs-mapped drag captions, z-order restoration, icon hydration retries, and Chinese IME support during item rename.
- Added clearer troubleshooting guidance for drag/drop failures caused by running DeskBox as administrator.
- Refined first-run onboarding with shorter Windows-style copy and simpler setup choices.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Quick Capture data, and managed storage content.

## 中文

DeskBox 1.0.9 是一次集中优化设置、格子交互、随记体验和日常桌面行为的版本。这个版本重点让 DeskBox 更接近 Windows 原生体验，也让托盘、快捷键、拖拽、右键菜单和随记之间的状态更稳定。

安装器会检测 .NET 8 Runtime x64 和 Windows App Runtime 2.1.3 x64。缺少依赖时，安装流程可以联网下载并安装。

如果资源管理器拖拽到 DeskBox 时仍然显示禁止符号，或无法把文件拖入格子，请先检查 DeskBox 是否被设置成“以管理员身份运行”。日常使用时，DeskBox 应与资源管理器保持普通用户权限；如果 DeskBox 以管理员权限运行，Windows 可能会在更底层阻止文件拖放。

### 主要更新

- 重构设置页面结构，统一使用原生 ComboBox、NumberBox、带“开 / 关”文字的开关、钻入式设置项、托盘图标选项和更清晰的随记设置入口。
- 新增格子内排序方式：名称、大小、项目类型、修改日期，并支持按格子保存排序状态和重复点击切换升序 / 降序。
- 优化格子右键菜单，让标题栏的格子管理操作和内容区的文件操作分工更清楚。
- 优化随记 Tab 切换、标题栏按钮、悬浮按钮、复制反馈，以及与普通格子一致的显示 / 恢复层级行为。
- 增强拖拽兼容、空格子拖放、收纳 / 映射拖拽提示、层级恢复、图标加载重试和文件重命名中文输入法支持。
- 补充拖拽异常排查说明：DeskBox 不应以管理员权限运行，否则普通权限资源管理器可能无法把文件拖入格子。
- 精简新用户引导文案和设置选项，更贴近 Windows 风格。

### 兼容性

- 目标系统：Windows 11。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、格子配置、随记数据和收纳目录内容。
