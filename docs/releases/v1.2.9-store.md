## Store Description

### English

DeskBox adds a focused organization layer to the Windows desktop without replacing the desktop shell. Create file widgets backed by real folders, map existing folders without moving their contents, and keep frequently used information close at hand.

Use Todo for card-based tasks with due dates, reminders, recurrence, color markers, attachments and batch actions. Use Quick Capture for text and image notes with paper styles, pinning and drag sorting. Responsive Weather and Music widgets adapt to compact desktop layouts, while tray controls and a global hotkey make all widgets easy to show, hide or raise.

DeskBox is built with WinUI 3 and Windows App SDK, follows Windows 11 interaction patterns, and keeps user data local. File widgets continue to use normal Windows files and folders.

### 中文

DeskBox 在不替换 Windows 桌面的前提下，为桌面增加一层克制的整理能力。你可以创建由真实文件夹支撑的收纳格子，也可以映射已有文件夹而不移动其中内容，把常用文件和信息放在更容易访问的位置。

待办格子支持卡片任务、截止日期、提醒、重复、颜色标记、附件和批量操作；随记支持文字与图片、纸张样式、固定和拖动排序；天气与音乐格子会根据尺寸自动调整布局。通过托盘和全局快捷键，可以随时显示、隐藏或唤起全部格子。

DeskBox 基于 WinUI 3 与 Windows App SDK 构建，遵循 Windows 11 的设计和交互习惯，用户数据保存在本地，文件格子使用的仍然是普通 Windows 文件与文件夹。
