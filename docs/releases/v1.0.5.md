# DeskBox 1.0.5

## English

DeskBox 1.0.5 focuses on first-run clarity, daily access, and safer storage maintenance. This release rebuilds onboarding with a stronger logo intro and feature walkthroughs, then adds an optional global hotkey so showing, hiding, and temporarily raising widgets feels faster.

### Download

- [DeskBox_Setup_1.0.5_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.5/DeskBox_Setup_1.0.5_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Rebuilt onboarding: shows the DeskBox logo motion first, then moves smoothly into a five-step guide.
- Improved onboarding scenes: feature demos now loop and better match creating widgets, dropping files, choosing move/copy behavior, mapping folders, and daily access.
- Global hotkey: enable an optional shortcut for the same show, hide, and temporary-raise flow as tray left-click.
- Better Settings and tray access: added managed-storage opening, Quick Access pinning, download-link actions, and clearer maintenance controls.
- Better storage maintenance: improved default storage migration, mapped shortcut sync, orphan managed-folder cleanup, and safety prompts for destructive actions.
- Cleaner resource release: removed stale blur-toggle leftovers and releases animation/window references more promptly after Settings or onboarding closes.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, and managed storage content.

## 中文

DeskBox 1.0.5 主要优化首次使用体验、日常唤起方式和文件收纳维护流程。这个版本重构了新用户引导，加入了更完整的品牌动效和步骤演示，同时新增全局快捷键，让格子的显示、隐藏和临时置顶更顺手。

### 下载

- [DeskBox_Setup_1.0.5_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.5/DeskBox_Setup_1.0.5_x64.exe)

安装器会检测 .NET 8 Runtime x64 和 Windows App Runtime 2.1.3 x64。缺少依赖时，安装流程可以联网下载并安装。

### 主要更新

- 新用户引导重构：前置展示 DeskBox logo 动效，再自然进入五步引导流程。
- 引导动效优化：步骤演示支持循环播放，右侧场景更贴合“创建格子、拖入文件、移动/复制、映射文件夹、日常唤起”等操作。
- 全局快捷键：可在设置中启用，用键盘触发与托盘左键一致的显示、隐藏和临时置顶流程。
- 设置与托盘入口优化：增加打开默认收纳目录、固定到快速访问、下载链接和维护相关操作。
- 文件收纳维护优化：支持默认收纳路径迁移、映射快捷方式同步、孤立收纳目录清理，并强化危险操作提示。
- 资源释放优化：清理旧的模糊开关残留；设置窗口和新用户引导关闭后会更及时释放动画与窗口引用。

### 兼容性

- 目标系统：Windows 11。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、组件配置和收纳目录内容。
