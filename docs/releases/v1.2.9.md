# DeskBox 1.2.9

DeskBox 1.2.9 focuses on the widgets used every day. Todo and Quick Capture now use card lists with focused detail editing, Music and Weather adapt to smaller widget sizes, and file/menu/title interactions are more consistent across the app. This release also tightens reset behavior, dynamic Z-order, memory cleanup and cross-widget refresh performance.

## Highlights

- Todo opens tasks in a dedicated full-widget detail view and adds compact scheduling, eight color markers, attachments, drag sorting and batch commands.
- Quick Capture adds paper styles, image add/replace/preview/copy, pinning, optional creation times, animated sorting and batch actions.
- Music and Weather adapt down to 150 x 150. Music uses centered artwork cropping, title marquee, compact playback controls and smoother artwork transitions.
- All music spectrum visuals and settings have been removed for a simpler, lighter widget.
- File widgets add multi-selection actions; File, Todo and Quick Capture context commands now close their menus immediately.
- Tray/hotkey raising and dynamic Z-order are more predictable, and widget title rename now focuses immediately with double-click support.
- Reset behavior now clearly separates preference reset, feature-widget data reset and feature-widget enable/disable state.
- Reduced unrelated widget refreshes and improved cleanup during repeated music, language, theme and material changes.

### 更新重点

- 待办点击后进入格子内完整详情页，支持紧凑时间设置、8 种颜色标记、附件、拖动排序和批量操作。
- 随记支持纸张样式、添加/替换/预览/复制图片、固定、创建时间显示、带动效排序和批量操作。
- 音乐和天气支持最低 150 x 150；音乐使用封面居中裁剪、歌名跑马灯、紧凑控制和平滑封面切换。
- 完整移除音乐频谱功能与相关设置，降低无意义的持续资源占用。
- 文件格子增加多选操作，文件、待办和随记的右键命令执行后立即关闭菜单。
- 托盘/快捷键唤起与动态层级更稳定；标题重命名立即聚焦，并支持双击标题编辑。
- 明确区分偏好设置重置、功能格子数据重置和功能格子开关，避免误清除数据。
- 减少无关格子刷新，完善音乐、语言、主题和材质反复切换时的资源释放。

## Test Checklist

- Verify Direct About shows `1.2.9` and the Store build shows `1.2.9.0`.
- Test fresh install and overwrite install; existing settings, widgets and managed files must remain intact.
- Exercise Todo and Quick Capture creation, detail return, drag sorting, box selection, context menus and batch commands.
- Verify Quick Capture image add, replace, preview and copy after switching tabs and restarting.
- Resize Music and Weather through 150 px, 180 px and medium layouts; change songs repeatedly and confirm artwork never flashes empty.
- Trigger tray and hotkey show/hide while other windows cover some widgets; confirm all widgets raise and later follow normal Windows Z-order.
- Repeatedly change language, theme, material, opacity and corner style and observe responsiveness and memory behavior.
- Verify global reset preserves language, startup and feature-widget switches; verify each feature-widget reset asks for confirmation and clears only that widget's data/settings.
