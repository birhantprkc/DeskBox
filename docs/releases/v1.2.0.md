# DeskBox 1.2.0

## English

DeskBox 1.2.0 is the first large release after 1.1.10. It includes a broad widget architecture refactor, introduces the feature-widget foundation, adds Todo and Music as first-class desktop widgets, modernizes Quick Capture, and reorganizes Settings around the new model. This release is larger than the usual maintenance builds because a lot of internal widget plumbing was rebuilt to make future widgets easier to add and maintain.

### Highlights

- Rebuilt the widget architecture around shared shell, content host, content factory, registry, session, positioning, diagnostics, and window-factory services.
- Added the feature-widget foundation used by Todo, Quick Capture, Music, and future content widgets.
- Added the Todo widget with local task storage, completion state, filters, inline editing, full-screen editing, and custom due times.
- Added the Music widget with Windows media session integration, playback controls, playback mode switching, system volume control, responsive waveform styles, compact layout, and optional album-color ambience.
- Migrated Quick Capture onto the newer widget/content infrastructure, improved editing consistency, and added cached thumbnails for recent image previews.
- Unified widget chrome and editing details across file and feature widgets: title bars, title styles, inline editors, full-screen editors, hover/pressed states, empty states, tooltips, and action controls.
- Reorganized Settings for the new architecture, including feature widgets, file-widget display, appearance, interaction/global hotkey, Quick Capture, and music options.
- Expanded automated tests around widget factories, registry/session/positioning, content windows, Todo, feature-widget settings, storage cleanup, and Quick Capture image handling.
- Changed the project license to GPL-3.0-only for future source code and releases. Previously published MIT-licensed versions remain under MIT.

### Notes For Testing

- Test overwrite install from 1.1.10 and confirm existing widgets, files, Quick Capture data, Todo data, settings, and managed storage content remain intact.
- Test a fresh install on another Windows 11 machine, especially first launch, tray startup, global hotkey, file drag/drop, Todo, Quick Capture recent images, and Music.
- If Windows App Runtime or .NET 8 Runtime is missing, verify that the installer dependency prompt appears correctly.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Todo data, Quick Capture data, and managed storage content.

### 主要更新

- 重构格子架构：文件格子和功能格子开始共享外壳、内容宿主、内容工厂、注册表、会话、定位、诊断和窗口工厂等基础服务。
- 建立功能格子基础设施，用于承载待办、随记、音乐以及后续更多内容格子。
- 新增待办格子：支持本地任务存储、完成状态、筛选、行内编辑、全屏编辑和自定义结束时间。
- 新增音乐格子：接入 Windows 媒体会话，支持播放控制、播放模式切换、系统音量控制、自适应频谱、紧凑布局和可选封面氛围取色。
- 随记迁移到新的格子/内容体系，编辑体验更一致，并为最近图片预览增加缩略图缓存，减少图片较多时的卡顿和内存压力。
- 统一文件格子和功能格子的外壳与编辑细节：标题栏、标题样式、行内编辑、全屏编辑、悬停/按下状态、空状态、tooltip 和操作控件。
- 按新架构重新整理设置页，包括功能格子、文件格子显示、外观、交互/全局快捷键、随记和音乐相关设置。
- 扩展自动化测试覆盖：新增格子工厂、注册/会话/定位、内容窗口、待办、功能格子设置、收纳清理和随记图片处理相关测试。
- 项目授权协议调整为 GPL-3.0-only，适用于后续源码和版本；此前已经按 MIT 发布的旧版本仍保持 MIT 授权。

### 测试建议

- 建议在另一台 Windows 11 电脑上测试全新安装，重点看首次启动、托盘、开机自启、全局快捷键、文件拖拽、待办、随记最近图片和音乐格子。
- 如果目标电脑缺少 Windows App Runtime 或 .NET 8 Runtime，请确认安装器能正确弹出依赖安装提示。

- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
