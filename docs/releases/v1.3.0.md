# DeskBox 1.3.0

DeskBox 1.3.0 introduces a new compact desktop workflow. Widgets can collapse into information-rich capsules, form an ordered combined bar, and expand from stable screen-aware anchors. File widgets gain automatic stacks, while Settings, attachment workflows and data recovery have been rebuilt for clearer daily use and safer upgrades.

## Highlights

- Collapse widgets into compact capsules with click or title-area hover expansion, smart/summary/minimal content and optional privacy protection.
- Keep capsule and expanded geometry aligned or adjust them independently. Screen-aware anchoring, drag-to-expand, per-widget overrides and resize guides keep both states stable.
- Arrange capsules independently or combine them into an ordered movable bar with floating or edge placement, automatic direction and adjustable spacing.
- Automatically stack file-widget items by type or date, or build prioritized custom extension rules with live previews and an unmatched-file policy.
- Press Space to preview a selected file through QuickLook when QuickLook is already running; DeskBox adds no hard dependency or integration settings.
- Attach multiple files to Todo and Quick Capture entries, choose link or managed-copy storage, and include localized attachment paths when copying content.
- Configure Todo and Quick Capture tabs, drag entries onto tabs to change state, tune list preview line counts, and choose Enter or Ctrl+Enter for saving.
- Use Mica Alt, Base acrylic, material intensity, border color modes and display-density presets. Music can stay adaptive or force Cover/Controls mode.
- Navigate a Windows-style Settings hierarchy with title-bar search, focused detail pages and important options surfaced on their entry cards.
- Export and restore integrity-checked backups, manage automatic snapshots, recover resilient data stores and scan attachment health.
- Overwrite installs now close a running DeskBox process reliably before replacing files, avoiding the intermittent Retry / Ignore / Cancel prompt.

## 中文说明

DeskBox 1.3.0 带来新的桌面紧凑工作流。格子可以收起为包含关键信息的胶囊，也可以组成可排序的组合栏，并从稳定、感知屏幕边界的锚点展开。文件格子新增自动叠放；设置、附件和数据恢复流程也进行了系统重构，让日常使用更清楚，版本升级更安全。

### 更新重点

- 格子可收起为胶囊，支持点击或仅标题区域悬停展开，并可选择关键信息、简要摘要、仅图标和标题以及隐私保护。
- 胶囊与展开状态可以保持同宽，也可以独立调整；屏幕边界锚点、拖入临时展开、单格覆盖和调整大小参考线共同保证位置稳定。
- 胶囊可独立摆放或组成能够整体移动的组合栏，支持自定义顺序、悬浮/贴边位置、自动方向和间距。
- 文件格子可按类型或日期自动叠放，也可创建带优先级、实时预览和未匹配策略的自定义扩展名规则。
- QuickLook 已运行时，可在选中文件后按空格预览；DeskBox 不增加强制依赖或额外集成设置。
- 待办和随记支持多个附件、关联原路径或复制到 DeskBox，并在复制正文时按当前语言附带附件路径。
- 待办和随记可配置标签页，内容可拖到标签页直接改变状态；列表预览行数和 Enter/Ctrl+Enter 保存方式均可调整。
- 新增云母 Alt、标准亚克力、材质浓度、边框颜色模式和显示密度预设；音乐可保持自适应或强制封面/控制模式。
- 设置改为更接近 Windows 的层级与标题栏搜索，主要选项直接放在入口卡片，详细页面只保留高级配置。
- 支持带完整性校验的备份导出与恢复、自动快照、弹性数据恢复和附件健康检查。
- 覆盖安装会在替换文件前可靠关闭正在运行的 DeskBox，避免偶发出现“重试 / 忽略 / 取消”提示。

## Test Checklist

- Verify About shows `1.3.0`, then test fresh install and overwrite install on a clean Windows 11 x64 PC.
- Test with .NET 10 and Windows App Runtime 2.2 already installed, then test a machine missing one dependency and confirm setup downloads only what is missing.
- Confirm existing widgets, positions, compact/expanded sizes, managed folders, Todo, Quick Capture and attachments survive an overwrite upgrade from 1.2.9.
- Exercise click and hover capsule modes, aligned and independent widths, each screen edge, combined bar order, drag-to-expand and rapid repeated expand/collapse.
- Test file stacks with type, date and custom rules; rapidly switch stacks and verify no intermediate spread state remains stuck.
- If QuickLook is installed and running, select files and press Space repeatedly. Without QuickLook, Space must have no DeskBox error or startup side effect.
- Test Todo and Quick Capture multiple attachments in link and copy modes, localized copy output, tab drag actions, hidden tabs, preview lines and both Enter-key modes.
- Export a backup, inspect snapshot inventory, restore after restart and run the attachment health scan.
- Change language while Settings is open and verify combo-box values, search results, breadcrumbs and entry-card controls remain visible and selected.
- Test F7/tray show and hide, animation styles, tray menu height, multi-monitor/DPI changes and display reconnects.
