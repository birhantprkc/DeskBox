# DeskBox 1.2.7

## English

DeskBox 1.2.7 is a performance-focused release. It improves animation smoothness on high-refresh-rate displays, reduces hover and backdrop refresh overhead, and lowers memory usage from icon caching.

### Highlights

- Widget tray animations now use `CompositionTarget.Rendering` (VSync-synced) instead of a fixed 16ms timer, delivering native-refresh-rate animation on 60Hz, 120Hz, and 144Hz displays.
- Backdrop refresh logic replaced three `Task.Delay` calls with a single staged `DispatcherQueueTimer`, reducing thread-pool pressure.
- Removed an empty `PointerMoved` handler from widget item surfaces and added state guards to hover handlers, reducing unnecessary work during window close, hide animation, or invisible states.
- Music widget timers (progress, visualizer, transition) are now stopped when the widget is deactivated and restarted on activation, eliminating idle CPU usage.
- System theme/accent color changes are debounced by 200ms in `ThemeService`, preventing redundant appearance refresh cascades.
- Icon cache limit reduced from 500 to 200 entries, and `DecodePixelWidth` is now set before `SetSourceAsync` for both icons (48px) and image thumbnails (80px), lowering decoded bitmap memory.

### Testing Notes

- Verify widget show/hide animations are smooth on 60Hz and high-refresh-rate displays.

### 更新重点

- 格子托盘动画改用 `CompositionTarget.Rendering`（VSync 同步）替代固定 16ms 定时器，在 60Hz、120Hz、144Hz 显示器上均以原生刷新率运行。
- 毛玻璃背景刷新逻辑将三次 `Task.Delay` 替换为单个分阶段 `DispatcherQueueTimer`，减少线程池压力。
- 移除格子项目表面的空 `PointerMoved` 处理器，并为悬停处理器增加状态守卫，减少窗口关闭、隐藏动画或不可见状态下的无效计算。
- 音乐格子定时器（进度、频谱、过渡）在失焦时停止、激活时重启，消除空闲 CPU 占用。
- 系统主题/强调色变化在 `ThemeService` 中增加 200ms 防抖，避免冗余的外观刷新级联。
- 图标缓存上限从 500 降至 200，并在 `SetSourceAsync` 之前设置 `DecodePixelWidth`（图标 48px、缩略图 80px），降低解码位图内存。

### 测试建议

- 在 60Hz 和高刷新率显示器上检查格子显示/隐藏动画是否流畅。
- 快速移动鼠标划过多个格子项目时，检查悬停效果是否无卡顿。
- 主题切换、显示器拓扑变化、格子显示/隐藏后，检查毛玻璃背景是否正确渲染。
- 播放音乐时检查频谱动画，失焦后确认定时器已停止。
- 切换包含大量文件的文件夹后，检查图标加载和缩略图预览是否正确。
- 确认 Direct 版关于页显示 1.2.7，Store 包版本为 1.2.7.0。
