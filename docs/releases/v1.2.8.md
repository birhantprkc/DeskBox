# DeskBox 1.2.8

## English

DeskBox 1.2.8 is a major feature release. It introduces the Weather widget, adds a resize snap guide for precise widget alignment, fixes Quick Capture data clearing and tab switching issues, unifies default settings across new installs and resets, and completes a significant code refactoring to improve maintainability.

### Highlights

- **Weather widget**: A full-featured weather widget with four layout modes (Mini, Compact, Standard, Detailed). Supports offline city search with `cities.json` database and Windows Location API, auto-location, hourly and weekly forecasts, sunrise/sunset, UV index, precipitation, humidity, wind, pressure, rich skin backgrounds, and configurable refresh interval. Standard view activates at 250×215 and above. Horizontal scrollbar and mouse wheel support for the hourly temperature list. Week forecast fills remaining space and is scrollable even on small widgets. Loading animation with rotating refresh icon during data refresh.
- **Resize snap guide**: When resizing a widget, alignment guides highlight edges that match other widgets or the work area boundary (8px threshold), showing accent-colored highlight bars on both the resizing and target widget edges. Integrated into all widget window types.
- **Quick Capture fixes**: Resetting the Quick Capture widget now properly clears all internal data. Fixed blank display on first tab switch — data appears immediately instead of requiring multiple switches.
- **Settings defaults unified**: Default appearance is now Mica material, Medium border, Round corners — consistent across new installs, reset-to-defaults, and `AppSettings` initial values. Global reset now restores `CustomAccentColor` and `FocusClickedWidgetOnRaise`. Todo widget reset restores reminder settings (`TodoReminderEnabled`, `TodoDefaultReminderOffsetMinutes`). Weather widget reset clears saved coordinates (`WeatherLatitude`, `WeatherLongitude`).
- **Widget title font size**: Content widget titles now dynamically follow the global text size setting instead of using a fixed size.
- **Memory leak fix**: `TodoWidgetViewModel` and `TodoWidgetContentAdapter` now implement `IDisposable` to prevent leaks on widget disposal.
- **Delete widget button**: Added delete widget option to the "more" menu for all widget types.
- **Settings UI expansion**: Added standalone material type selector (Mica / Acrylic / Solid) and border style selector (None / Thin / Medium) to the Appearance settings page.
- **Widget layer improvement**: Added `SetWindowToDesktopLevel` method to push windows to desktop level without `HWND_BOTTOM`, preventing widgets from being hidden by Win+D.
- **Code refactoring**: Extracted `WidgetWindowBase.cs` (1027 lines) as shared base for all widget windows. Split `WidgetWindow.xaml.cs` (~5000 lines) into 6 partial classes. Split `WidgetManager.cs` (3532 lines) into 4 partial classes. Log rotation added (5MB threshold with backup). Settings writes are now atomic via temp-file-and-move pattern. `FolderWatcherService` debounce migrated to `DispatcherQueueTimer`. Shortened drag-and-drop diagnostics text for clearer messaging.

### 更新重点

- **天气格子**：全新功能天气格子，支持四种布局模式（迷你、紧凑、标准、详细）。支持离线城市搜索（`cities.json` 数据库 + Windows Location API）、自动定位、逐小时和每周预报、日出日落、紫外线、降水、湿度、风速、气压、丰富皮肤背景和可配置刷新频率。标准视图在 250×215 及以上尺寸激活。逐小时温度列表支持横向滚动条和鼠标滚轮。周预报填满剩余空间，小尺寸下也可滚动。刷新时显示旋转加载动画。
- **调整大小参考线**：调整格子大小时，对齐参考线高亮匹配的边缘（8px 阈值），与其他格子边缘和工作区边界对齐时显示强调色高亮条。已集成到所有格子窗口类型。
- **随记修复**：重置随记格子现在正确清理所有内部数据。修复首次切换 Tab 时空白显示的问题——数据立即显示，无需多次切换。
- **默认设置统一**：默认外观改为云母材质、中等边框、大圆角——新安装、恢复默认和初始值三处一致。全局重置现在恢复 `CustomAccentColor` 和 `FocusClickedWidgetOnRaise`。待办格子重置恢复提醒设置（`TodoReminderEnabled`、`TodoDefaultReminderOffsetMinutes`）。天气格子重置清除保存的坐标（`WeatherLatitude`、`WeatherLongitude`）。
- **格子标题字号**：功能格子标题现在动态跟随全局文字大小设置，不再使用固定字号。
- **内存泄漏修复**：`TodoWidgetViewModel` 和 `TodoWidgetContentAdapter` 实现 `IDisposable`，防止格子销毁后内存泄漏。
- **删除格子按钮**：所有类型格子的"更多"菜单中均可删除格子。
- **设置页扩展**：外观设置新增独立的材质类型选择（云母 / 亚克力 / 纯色）和边框样式选择（无边框 / 细 / 中）。
- **格子层级改进**：新增 `SetWindowToDesktopLevel` 方法，不使用 `HWND_BOTTOM` 即可将窗口推至桌面层级，防止 Win+D 隐藏格子。
- **代码重构**：提取 `WidgetWindowBase.cs`（1027 行）作为所有格子窗口的共享基类。`WidgetWindow.xaml.cs`（约 5000 行）拆分为 6 个 partial 类。`WidgetManager.cs`（3532 行）拆分为 4 个 partial 类。新增日志轮转（5MB 阈值自动备份）。设置写入改为原子操作。`FolderWatcherService` 防抖迁移至 `DispatcherQueueTimer`。精简拖拽诊断文案。
