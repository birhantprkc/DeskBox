# DeskBox 1.0.7

## English

DeskBox 1.0.7 focuses on stability and polish after the Quick Capture release. This version tightens the shared widget layering model, makes tray and global-hotkey behavior more predictable in full-screen apps, improves widget animations, and refines Quick Capture into a more comfortable daily-use surface for copying, opening, editing, and managing saved content.

### Download

- [DeskBox_Setup_1.0.7_x64.exe](https://github.com/Tianyu199509/DeskBox/releases/download/v1.0.7/DeskBox_Setup_1.0.7_x64.exe)

The installer checks for .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64. If either dependency is missing, setup can download and install it.

### Highlights

- Unified widget layering: file widgets and Quick Capture now raise, hide, and restore as one DeskBox group from the tray or global hotkey.
- Better full-screen behavior: F7 can raise widgets again when they are visible but covered, and a keyboard-hook fallback helps DeskBox win the configured hotkey before apps such as Axure consume it.
- Smoother animations: widget show/hide uses linear timing, a shorter default duration, and group-aware off-screen slide distances.
- Better Quick Capture workflow: click copies with feedback, double-click opens text, links, or images in the default system app, and long text editing now uses a compact inline editor.
- Cleaner Quick Capture UI: wider text rows, right-aligned hover actions, clearer action styling, faster tab switching, improved image previews, and theme-aligned edit surfaces.
- Storage recovery: Settings now includes orphan managed-storage folder management for restoring, opening, moving back, or deleting folders left by removed widgets.
- Safer storage and mapping: duplicate-name guards, startup recovery for bad rename states, mapped-folder rename sync, icon refresh improvements, and file-name display fixes.
- Responsiveness fixes: reduced list/tab animation cost, improved clipboard recording behavior, steadier directory refresh, and stronger temporary-topmost confirmation.

### Compatibility

- Target system: Windows 11.
- Runtime dependencies: .NET 8 Runtime x64 and Windows App Runtime 2.1.3 x64.
- Overwrite installs preserve existing app settings, widget configuration, Quick Capture data, and managed storage content.

DeskBox 1.0.7 是随记发布后的稳定性和体验打磨版本。这个版本重点收口格子层级、托盘和全局快捷键逻辑，优化全屏应用中的唤起表现，调整动画节奏，并让随记在复制、打开、编辑和内容管理上更适合日常使用。

### 主要更新

- 统一格子层级：文件格子和随记现在通过托盘或全局快捷键唤起时，会作为同一个 DeskBox 组统一置顶、隐藏和恢复。
- 优化全屏应用体验：格子可见但被外部应用遮挡时，F7 会直接重新置顶；并加入快捷键钩子兜底，避免 Axure 等应用先消费快捷键。
- 动画更顺滑：格子显示/隐藏改为线性节奏、更短默认时长，并按整组位置计算滑出屏幕距离。
- 随记流程优化：单击复制并提示成功，双击用系统默认应用打开文本、链接或图片，长文本编辑改为紧凑的内部编辑层。
- 随记界面优化：记录文字区域更宽，悬浮按钮右对齐，按钮状态更清晰，Tab 切换更快，图片预览和编辑层样式更贴合主题。
- 收纳目录恢复：设置中新增孤立收纳文件夹管理，可恢复、打开、移回桌面或删除被移除格子留下的文件夹。
- 收纳和映射更稳：增加重名保护、异常重命名状态恢复、映射文件夹改名同步、图标刷新稳定性和文件名显示修复。
- 响应性优化：减少列表和 Tab 动画成本，优化剪贴板记录、目录刷新和临时置顶确认，降低偶发卡顿和漏置顶。

### 兼容性

- 目标系统：Windows 11。
- 运行依赖：.NET 8 Runtime x64、Windows App Runtime 2.1.3 x64。
- 覆盖安装会保留现有应用设置、组件配置、随记数据和收纳目录内容。
