# DeskBox 1.3.1

DeskBox 1.3.1 is a quality and performance release. It fixes icon clarity on high-DPI displays, resolves Steam shortcut icons, improves animation smoothness, fixes a black-screen transition, unifies startup entries, and updates capsule mode defaults for a better first-run experience.

The installer is framework-dependent. It does not bundle the .NET 10 or Windows App Runtime installers; it checks the target PC and downloads only a missing dependency.

## Highlights

- File and shortcut icons now extract at 256×256 (Jumbo) or 48×48 (Extra Large) instead of 32×32, matching desktop icon clarity on high-DPI displays.
- Resolved `.url` shortcut icons not displaying by parsing comma-separated icon indices, searching for Steam via registry, and falling back to common install locations.
- Switched opacity/scale animations to GPU Composition KeyFrame animations and direct `SetWindowPos` P/Invoke for lower latency and smoother playback.
- Material backdrop now stays consistent throughout tray show/hide animations — no more black-screen transition.
- Fixed asymmetric corner radius on music widget album art in both empty and populated states.
- Unified startup registration to a single registry `Run` key with automatic cleanup of legacy shortcuts.
- New installs and global reset now default to hover auto-expand, sensitive hover response, and relaxed expand/collapse animation.
- Added a UI thread watchdog for freeze diagnostics and fixed a sync-over-async deadlock in Store startup.

## 中文说明

DeskBox 1.3.1 是一个质量和性能版本。修复了高 DPI 显示器上的图标清晰度、Steam 快捷方式图标不显示、动画卡顿、黑屏过渡、音乐封面圆角不对称和重复启动项问题，并更新了胶囊模式默认值。

### 更新重点

- 文件和快捷方式图标现在提取 256×256（Jumbo）或 48×48（Extra Large），在高 DPI 显示器上与桌面图标一样清晰。
- 修复 `.url` 快捷方式图标不显示的问题，解析逗号分隔的图标索引，通过注册表和常见路径搜索 Steam。
- 将透明度/缩放动画切换为 GPU Composition KeyFrame 动画和直接 `SetWindowPos` P/Invoke，更流畅、延迟更低。
- 材质背景在托盘显示/隐藏动画期间保持一致 — 不再出现黑屏过渡。
- 修复音乐格子封面在空状态和有值时圆角不对称的问题。
- 统一开机启动注册为单一注册表 `Run` 键，自动清理旧快捷方式。
- 新安装和全局重置现在默认使用悬停自动展开、灵敏悬停响应和舒缓动画。
- 新增 UI 线程看门狗用于卡死诊断，修复 Store 启动的 sync-over-async 死锁。

## Test Checklist

- Verify About shows `1.3.1`, then test fresh install and overwrite install on a clean Windows 11 x64 PC.
- On a high-DPI display (150%+ scaling), verify file and shortcut icons are crisp and match desktop icon clarity.
- Drag a Steam `.url` shortcut into a file widget and verify the icon displays correctly.
- Test tray show/hide animations (F7) with Fade, Slide, ScaleFade, and Zoom effects — no black-screen transition.
- Verify music widget album art has symmetric corner radius in both empty and populated states.
- Check Windows Settings → Startup — only one DeskBox entry should exist.
- New install should default to hover auto-expand, sensitive hover response, and relaxed animation in capsule mode.
- Test overwrite upgrade from 1.3.0 — existing widgets, settings, and managed storage should survive.
