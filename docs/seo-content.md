# DeskBox - 轻量级 Windows 11 桌面整理工具

## 简介（150字以内，用于搜索引擎摘要）

DeskBox 是一个基于 WinUI 3 的 Windows 11 桌面整理工具。用轻量桌面格子帮你收纳文件、映射文件夹、管理剪贴板。支持全局快捷键一键唤起、拖拽操作、图片缩略图预览、动画效果自定义。开源免费，采用 GPL-3.0-only（GNU GPL v3）协议（早期版本曾为 MIT，协议变更不追溯旧版）。

## 功能描述（SEO 优化版）

### 收纳格子
创建桌面格子，将文件分门别类整理。支持图标视图和列表视图，拖拽文件直接入格，支持排序、搜索、批量操作。图片文件自动显示缩略图预览。

### 文件夹映射
将已有文件夹直接映射为桌面格子，无需移动文件。保持原有文件夹结构，支持拖拽操作和右键菜单。

### 随记
本地剪贴板管理器，自动记录文本、链接和截图。支持搜索、置顶、保存到文件格子。全局快捷键 F7 一键唤起。

### 全局快捷键
默认 F7 快捷键，可自定义。全屏应用下也能唤起格子，支持拖拽内容到其他应用。

### 外观自定义
明暗主题、透明度、圆角样式、图标大小、文字大小、动画效果和速度，全部可调。

### 拖拽诊断
内置诊断工具，一键修复 Windows 11 拖拽兼容性问题。
